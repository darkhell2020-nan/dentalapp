# Professional illustration assets

Place professionally reviewed SVG illustrations in this folder or nested folders. DentVisual discovers SVGs here through the central illustration registry and matches them to card `illustrationId` values and common aliases, for example:

- `dental-crown.svg`
- `crown-cross-section.svg`
- `chrome-cobalt-denture.svg`
- `cocr-components.svg`
- `acrylic-vs-chrome-cobalt-denture.svg`
- `dental-implant.svg`
- `root-canal-treatment.svg`

SVGs are rendered with `object-contain` so they remain responsive without cropping or distortion and are bundled into the Vite/PWA asset graph.
